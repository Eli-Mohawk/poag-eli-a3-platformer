#Welcome to Ascent!

##Overview
- Ascent is a 2d platformer game where your goal is to climb up each level and make it to the top to win
- There are 2 types of platforms (stationary, and moving)
- there are spikes, which cause you to return to the start of the current level and lose one of your 5 lives
- if you fall to the bottom of a level, you are respawned on the level before it

##Controls
- A / Left arrow - move left
- D / Right arrow - move right
- W / Up arrow - jump
- Space - detection ability

##Endings
- There are 4 endings in the game
- 1. Game over - run out of lives or die
- 2. "Win" - complete level 10
- 3. "Error" - don't listen to the game and complete level 10
- 4. True End - complete level 0 then complete level 10 (must be game completion 1 or 2)
